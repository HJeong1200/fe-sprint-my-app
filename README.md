# ⭐️ My Todo App ⭐️

## 🌈 주요 기능

### ⏰ 시계

- 현재 시간을 확인할 수 있습니다. (한국 기준)

### 🌦 기상 현황

- 현재 날씨와 기온을 알 수 있습니다.
- 주의!) 위치 확인 권한을 꼭 확인해주세요.

### 📅 달력

- 오늘의 날짜를 확인할 수 있습니다.
- 아쉽게도 날짜를 확인하는 것 외에 다른 기능은 없습니다.

### 📌 To-do 리스트

- 할일 목록을 추가할 수 있습니다.
- '◀︎'를 클릭하여 간단한 메모를 추가할 수 있습니다.
